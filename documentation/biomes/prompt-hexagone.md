Génère moi une image qui contient 7 hexagones parfaits, la pointe vers le haut. 

Ils seront disposés comme ça :

```
* * *
* * *
*
```

Chaque hexagone doit contient une image décrite ci-dessous.
Le style doit être réaliste pour chaque image.

## Désert

Vue de dessus : dunes de sable, terre craquelée, rochers dispersés.
Couleurs : beige, tan, marron clair.

## Forêt

Vue de dessus : canopée dense, arbres verts, végétation luxuriante.
Couleurs : vert foncé, vert clair, marron pour les troncs.

## Océan

Vue de dessus : eau bleue profonde, vagues, écume
Couleurs : bleu, turquoise, blanc pour l'écume.

## Montagne

Vue de dessus : pics escarpés, falaises, pentes rocheuses, sommets enneigés.
Couleurs : gris, marron, blanc pour la neige.

## Glace

Vue de dessus : champs de glace, glaciers, neige.
Couleurs : blanc, bleu clair, cyan pâle.

## Volcanique

Vue de dessus : coulées de lave, roches volcaniques sombres, évents de vapeur.
Couleurs : noir, rouge foncé, orange, jaune pour la lave.

## Plaine

Vue de dessus : étendue plate et dégagée, herbes courtes ou prairie, éventuellement des buissons isolés.
Couleurs : vert clair, jaune pâle, marron léger pour les chemins ou la terre apparente.
