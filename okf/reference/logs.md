# Directory Logs

Every directory contains a `log.md`.

The `log.md` records the evolution of its knowledge domain.

Record significant events such as:

- document creation
- document updates
- document moves
- document renames
- document archival
- structural changes

Each entry should include:

- date
- affected document(s)
- short description of the change

The log is a concise history of the directory, not a detailed change diff.
