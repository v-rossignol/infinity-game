# Naming Conventions

## Directories

Use:

- lowercase
- kebab-case
- meaningful names

Examples:

```
security
human-resources
network-services
```

## Documents

Use descriptive filenames.

Examples:

```
password-policy.md
employee-handbook.md
project-charter.md
```

Avoid:

- spaces
- CamelCase
- abbreviations that are not widely understood
- duplicate names within the same directory

Names should remain stable whenever possible to preserve links.
