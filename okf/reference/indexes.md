# Directory Indexes

Every directory contains an `index.md`.

The `index.md` is the entry point for its knowledge domain.

## Responsibilities

An `index.md` should contain:

- purpose of the directory
- list of documents
- list of child directories
- short description of each entry
- links to related domains

## Maintenance

Whenever a document or subdirectory is:

- added
- removed
- renamed
- moved

the corresponding `index.md` shall be updated.

The index should always reflect the current contents of the directory.
